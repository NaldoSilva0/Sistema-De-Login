Sistema de Login com Logs em Python

Descrição:
Projeto básico em Python que permite criar uma conta, fazer login e registrar os horários de entrada e saída em um arquivo .txt. O código não salva os usuários permanentemente, sendo apenas para fins de estudo e prática.

Funcionalidades
Cadastro de usuário com confirmação de senha (temporário)
Login com limite de 5 tentativas
Registro de login e logout em Log_registro.txt

Menu interativo pós-login:
[1] Ver logs
[2] Deslogar

Como usar
Execute main.py
Cadastre seu usuário e senha
Faça login com as credenciais criadas
Use o menu para ver logs ou deslogar

Melhorias futuras
Salvar usuários permanentemente (JSON ou banco de dados)
Adicionar opções de alterar senha, excluir conta
Melhorar logs com data completa e usuário
